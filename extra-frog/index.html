What is Extra Frog?

Extra Frog is a Discord bot for fun stuff and global frog pets.
Buy pet store frogs with Frog Credits and feed them, otherwise they die.
Explore the rainforest for frogs to adopt.

What commands does Extra Frog have?

[user] must be an @mention

  • 🐸help : Gives you a link to this page.
  • 🐸setup : If not in Extra Frog's database, get added.
  • 🐸me : Tally your Frog Credits and frogs.
  • 🐸info [user] : Gets [user]'s Frog Credits and frog count
  • 🐸shop : Shows frogs and food you can buy, and a menu to select it.
  • 🐸feed [frog] : Feed a frog. You only need to feed them once a week.
  • 🐸changelog : What is new and the date it changed.
  • 🐸args [args] : Put any arguments and it will tell you what those arguments are. For testing

How does Extra Frog work?

Extra Frog is made using Node.js with Discord.js

Server control people info

If you add Extra Frog, setting its read messages access to a channel stops it from responding there.
Even though it wants admin privellages, it doesn't do anything with it.
