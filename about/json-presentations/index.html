About JSON Presentations

This is a thing I made because I like to make slideshows, and after I discovered that
P5.js has a function to easily convert JSON files into JavaScript objects, I knew what to make.

You can easily make and edit scripts and present them using the tools in the program.
You can add text, video, images, and more to the slides and make their backgrounds colors,
images, and videos.

Nothing you upload is given to me in any way because I don't want it to, (I really need to organize some folders), and also I
don't have enough server hardware to do it.
